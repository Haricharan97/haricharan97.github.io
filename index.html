Chapter 1: Functions

Composite Functions

Composition is applying one function to the result of another. Notation (f ∘ g)(x) is read as "f of g of x".

(f ∘ g)(x) = f(g(x))

Inverse Functions

The inverse f-1(x) reverses the action of f(x). Geometrically, the graph of an inverse is a reflection of the original function over the line y = x.

y | / y=x (Mirror Line) | / | / * (b, a) [Inverse Point] |/ +------------ x /| / | * (a, b) [Original Point]
Worked Example: Finding Inverse

Question: Find the inverse of f(x) = (2x + 1) / (x - 3).

1. Swap variables: x = (2y + 1) / (y - 3)

2. Multiply by denominator: x(y - 3) = 2y + 1

3. Expand: xy - 3x = 2y + 1

4. Group y terms: xy - 2y = 3x + 1

5. Factor y: y(x - 2) = 3x + 1

Answer: f-1(x) = (3x + 1) / (x - 2)

Chapter 2: Algebra

Logarithms

Logarithms allow us to solve for exponents. Key laws must be memorized for paper 1 (non-calculator).

loga(xy) = logax + logay
loga(x/y) = logax - logay
loga(xk) = k · logax
Base Change: logab = ln(b) / ln(a)

The Binomial Theorem

Used to expand (a + b)n.

General Term: Tr+1 = nCr · an-r · br
Worked Example: Finding a specific term

Question: Find the coefficient of x2 in the expansion of (2x + 3)4.

1. Identify parts: a = 2x, b = 3, n = 4. We need the term with x2.

2. Determine r: Since term has (2x)4-r, we need 4-r = 2, so r = 2.

3. Apply Formula: 4C2 · (2x)2 · (3)2

4. Calculate: 6 · 4x2 · 9

5. Simplify: 6 · 36 · x2 = 216x2

Answer: The coefficient is 216.

Chapter 3: Trigonometry

The Unit Circle

For any angle θ, the coordinate on the unit circle is (cos θ, sin θ).

90° (π/2) | (-1,0) | (0,1) 180° -----+----- 0° / 360° (2π) (π) | (1,0) | 270° (3π/2) (0,-1)

Double Angle Identities

sin(2θ) = 2 sinθ cosθ
cos(2θ) = cos2θ - sin2θ
= 2cos2θ - 1
= 1 - 2sin2θ
Worked Example: Trig Equation

Question: Solve sin(2x) = cos(x) for 0 ≤ x ≤ π.

1. Use Identity: 2sin(x)cos(x) = cos(x)

2. Rearrange (Do not divide by cos x!): 2sin(x)cos(x) - cos(x) = 0

3. Factor: cos(x) [2sin(x) - 1] = 0

4. Solve cases:
Case A: cos(x) = 0 → x = π/2
Case B: sin(x) = 1/2 → x = π/6, 5π/6

Answer: x = π/6, π/2, 5π/6

Chapter 4: Sequences & Series

Arithmetic vs Geometric

Type Rule nth Term (un)
Arithmetic Add difference d u1 + (n-1)d
Geometric Multiply ratio r u1 · rn-1

Sum to Infinity

Only exists for Geometric series where |r| < 1 (convergent).

S∞ = u1 / (1 - r)
Worked Example: Sigma Notation

Question: Calculate Σ (from n=1 to ∞) of 10(0.5)n.

1. Expand terms:
n=1: 10(0.5) = 5
n=2: 10(0.25) = 2.5
n=3: 1.25

2. Identify type: Geometric, u1=5, r=0.5.

3. Apply Formula: S = 5 / (1 - 0.5)

Answer: 10

Chapter 5: Introduction to Calculus

Differentiation (First Principles)

The derivative f'(x) is the gradient of the tangent to the curve.

f'(x) = lim (h→0) [ (f(x+h) - f(x)) / h ]
| / (Tangent Line) y | / | _/ (Curve) | _/ |/ +------------ x

Power Rule

If f(x) = axn
Then f'(x) = anxn-1
Worked Example: Finding Equation of Tangent

Question: Find the equation of the tangent to y = x2 + 3 at x = 1.

1. Find y-coordinate: y = (1)2 + 3 = 4. Point is (1, 4).

2. Find Gradient (Derivative): dy/dx = 2x.

3. Evaluate Gradient at x=1: m = 2(1) = 2.

4. Equation of Line: y - y1 = m(x - x1)
y - 4 = 2(x - 1)

Answer: y = 2x + 2
